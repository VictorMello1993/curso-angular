## Executando o projeto

1. Clone o repositório e abra o branch `projeto1`
```sh
git clone https://github.com/VictorMello1993/CursoAngular.git

git checkout projeto1
```

2. Abra VS Code e instale as dependências do projeto.
```sh
npm install
```

3. Execute o projeto
```sh
ng serve
```
